Conflict Check Automation

Built a fully automated conflict-of-interest detection system using Microsoft Power Platform, replacing a 3-day manual email review process with an instant automated pipeline. Real Estate Price Prediction DSS

ML-Powered Decision Support System for Property Valuation

Dec 2025 2.2M Listings 84% Accuracy

About the Project

Built a production-ready Decision Support System that processes 2.2 million U.S. property listings from Realtor.com to deliver real-time price forecasts with 83.96% R² accuracy — exceeding industry benchmarks for real estate valuation (70–85% range). Designed following the full CRISP-DM framework across all 6 phases.

Key Achievements

  • R² = 0.8396 on 330K unseen test properties; MAE $188,726
  • Compared Random Forest, XGBoost, and LightGBM; tuned best model via GridSearchCV (+2.06%)
  • Engineered 10 predictive features: price-per-sqft, room counts, geographic encodings
  • K-Means clustering (k=4) segmented market into Budget, Mid-Range, Premium, and Luxury tiers
  • Feature importance: Bathrooms 38.5%, House Size 35.2%, Bedrooms 12.9%

Technologies Used

Python Scikit-learn XGBoost LightGBM Pandas NumPy Matplotlib Seaborn Jupyter Notebook

Automating Video Editing

AI that Controls Video Editing with Natural Language

Mar 2026 Red Hat Hackathon Built in 6 Hours

About the Project

An AI-powered automation system that revolutionizes video editing by understanding natural language commands and physically controlling CapCut. Built during Red Hat's "The Open Accelerator" hackathon, this full-stack solution eliminates hours of manual clicking and menu navigation for content creators.

The Challenge

Content creators waste countless hours navigating through menus, scrolling for effects, and manually dragging elements. We built an AI agent that lets you simply speak your intent:

  • "Add a fade transition" → Done
  • "Change the music to something upbeat" → Done
  • "Split here and delete the second part" → Done
  • "Export the video" → Done

Technical Architecture

  • Full Stack: Three independent layers - React UI, FastAPI agent, and MCP server
  • Agentic Logic: ReAct-style agent with LangGraph for multi-step reasoning
  • MCP Integration: FastMCP protocol with dynamic tool discovery
  • Streaming: Real-time SSE for token-by-token responses
  • CapCut automation with computer vision control

Technologies Used

React + Vite FastAPI LangGraph Google Gemini FastMCP MCP Protocol Red Hat AI SSE Python
